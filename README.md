# Proyecto Multimedia UMSA - Página Web

Esta carpeta contiene una página web estática lista para publicar en GitHub Pages.

## Estructura

```text
/
├── index.html
├── styles.css
├── script.js
├── assets/
│   ├── img/
│   └── video/
└── downloads/
```

## Cómo publicar en GitHub Pages

1. Crear un repositorio en GitHub, por ejemplo: `proyecto-multimedia-umsa`.
2. Subir todos los archivos de esta carpeta al repositorio.
3. Entrar a `Settings` del repositorio.
4. Entrar a `Pages`.
5. En `Build and deployment`, seleccionar:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Guardar.
7. Esperar unos minutos y abrir el enlace que GitHub Pages genere.

## Qué contiene la web

- Presentación general del proyecto.
- Actividad grupal UMSA Digital.
- Galería de diagramas BPM y flujo.
- Actividad individual de clasificación de texturas.
- Filtro de suavizado 3x3.
- Cover animado de “La Vaca Lola”.
- Descarga de informe, notebook, resultados y proyectos.

## Recomendación

Si el video pesa mucho, se puede dejar solo el ZIP de resultados en `downloads/` y reemplazar el video por un enlace a Google Drive.
